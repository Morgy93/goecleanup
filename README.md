## Über GöCleanup

GöCleanup ist eine Initiative, die sich für ein sauberes Göttingen einsetzt.
Wir organisieren regelmäßig Cleanups, bei denen wir Müll in Göttingen einsammeln.
Jeder kann mitmachen!
